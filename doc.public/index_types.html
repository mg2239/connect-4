Index of types

C
color [Board]

color represents the colors of a disk.

command [Command]

The type command represents a player command that is decomposed into a verb, and possibly a position.

P
position [Command]

The type position represents a position on the grid with column number and the color the player is using.

R
result [State]

The type for the result of a move.

T
t [State]

The type for a game state.

t [Board]

t is the type of a board
