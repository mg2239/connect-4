module State: sig .. end

Representation of dynamic adventure state.

type t = {
   board : Board.t;
   current : Board.color;
}

The type for a game state.

type result =
| Legal of t
| Illegal

The type for the result of a move.

val init_state : t

init_state is the initial state of the game.

val board : t -> Board.t

board st is the board of the game state st. Example: board {board: Board.empty; current: R} is Board.empty. Raises: None.

  • Returns The board of the game state.
st : The game state.
val current_player : t -> Board.color

current_player st is the current player of the game state st. Example: current_player {board: Board.empty; current: R} is R. Raises: None.

  • Returns The current player of the game state.
st : The game state.
val next_player : t -> Board.color

next_player st is the next player of the game state st. Example: next_player {board: Board.empty; current: R} is B. Raises: None.

  • Returns The next player of the game state.
st : The game state.
val game_state : t -> bool

game_state st is whether a player has gotten four in a row in game state st. Example: game_state {board: b; current: R} where b contains a four in a row by B is true. Raises: None.

  • Returns true if there exists a winner in the game state. Otherwise false.
st : The game state
val go : int -> t -> result

go col st is r if attempting to place a disk in column col of the board in state st results in r. Example: go 0 {board: Board.empty; current: R} is Legal {board: b; current: B}, where b is the previous board but with a red game piece in column 0. Raises: None.

  • Returns If col is not full, return Legal st', where st' is the board with the new disk in col and the current player is the next player. Otherwise return Illegal.
col : The column that the piece is being placed in
st : The game state
